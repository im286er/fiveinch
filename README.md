# 配置

## nginx 配置
```
location / {
            
            index  index.html index.htm index.php l.php;
            
            if (!-e $request_filename) {
              rewrite  ^(.*)$  /index.php?s=/$1  last;
              break;
            }
            
            autoindex  off;
        }
```
